# Keypad
## 从0开始客制化键盘  
这个项目将从pcb绘制开始从0客制化出一把键盘  
## 键盘结构  
|  结构   | 备注  |
|  ----  | ----  |
| 外壳  | 亚克力或3D打印 |
| 定位板  | 碳纤维 |
| 键帽  | Pbt |
| Pbc  | - |
| 轴体  | - |
| 轴座  | - |
| 主控  | 32u4 |
| RGB  | - |
| 声音结构  | 夹心棉和底棉等 |
| 腕托  | 磁吸式 |
## 键盘需求  
- 分体式  
    - 打字区可彩拆分(主键区)  
    - 功能键区可与主键区拆分  
    - 数字键区可与主键区拆分  
    - 编辑区可与主键区拆分  
- 配列为xx键配列  
- 可控RGB  
- oled屏幕  
- 预留可拓展接口
